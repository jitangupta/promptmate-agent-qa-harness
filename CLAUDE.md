# Claude Code Instructions

Use `AGENTS.md` as the source of truth for this repository.

This project is intentionally agent-neutral so Codex, Claude Code, Claude Desktop, and other agentic systems can run the same PromptMate QA harness without diverging instructions.

Start every testing or documentation task by reading `AGENTS.md`.

## Propagating Patterns

When a new generalizable pattern is discovered and documented in `AGENTS.md`, write the product-agnostic version into `PATTERNS.md` in the sibling `agent-qa-harness` repository as part of the same task.
